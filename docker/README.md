# Preparando Banco de Dados PostgresSQL com Docker + Backup

Este template oferece um serviço docker para manter um container com banco de dados PostgreSQL gerado a partir de um dockerfile para atender demandas comerciais, contendo backup e restauração

Altere o arquivo [docker-compose.yaml](./docker-compose.yaml) com cuidado, para evitar problemas

Para saber mais sobre o template, [clique aqui](./database-postgres/README.md)
